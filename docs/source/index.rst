
widget_jsmol
=====================================

Version: |release|

A Jupyter widget to use Jmol/Jsmol.


Quickstart
----------

To get started with widget_jsmol, install with pip::

    pip install widget_jsmol

or with conda::

    conda install widget_jsmol
